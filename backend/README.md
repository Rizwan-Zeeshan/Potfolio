# Backend - Muhammad Rizwan Portfolio

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Configure environment variables in `.env`:
   ```
   PORT=5000
   MONGODB_URI=your_mongodb_uri
   NODE_ENV=development
   ```
3. Start the server:
   ```bash
   # Development mode
   npm run dev
   
   # Production mode
   npm start
   ```

## API Endpoints
- `POST /api/contact`: Submit contact form message.
