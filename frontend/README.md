# Frontend - Muhammad Rizwan Portfolio

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

## Technologies
- React (Vite)
- Tailwind CSS
- Framer Motion
- Lucide React
- Axios
